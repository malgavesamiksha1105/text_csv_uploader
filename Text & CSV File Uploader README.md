# Text & CSV File Uploader

This is a simple web application that allows users to upload and display 
.txt and .csv files directly in the browser.

## Features
- Upload .txt and .csv files
- Display file content instantly
- Simple and clean user interface
- Built using HTML, CSS, and JavaScript
- Backend-python(Flask)

## Technologies Used
- HTML
- CSS
- JavaScript (FileReader API)

## How to Use
1. Open the application in a browser.
2. Select a .txt or .csv file.
3. Click the Upload button.
4. The file content will be displayed on the screen.

## Purpose
This project is created as a beginner-friendly frontend practice project 
to understand file handling in web applications.
